# list-builder

A tool to manage and compile blocklists for Mastodon

## Install

```
git clone https://github.com/azcoigreach/mastodon-domain-blocklist.git
cd mastodon-domain-blocklist
pip install --editable .
```

## Usage

```
list-builder -help
```
